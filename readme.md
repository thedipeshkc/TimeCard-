# TimeCard — Work Hours Tracker

A professional, installable PWA for tracking work hours, calculating overtime, and estimating pay. All data stays on-device.
https://thedipeshkc.github.io/TimeCard-/

## Install on Mobile

### Android (Chrome)
After visiting the live URL, https://thedipeshkc.github.io/TimeCard-/
Chrome shows an **"Add to Home Screen"** banner automatically, 
or tap the menu (⋮) → **Add to Home Screen**.

### iOS (Safari)
1. Open the site in Safari
2. Tap the **Share** button (box with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **Add**

The app opens full-screen with no browser chrome, just like a native app.


## Features

- **Calculator** — Enter shifts manually, see overtime & pay breakdown
- **Clock In/Out** — One-tap timer that saves to history automatically
- **Time Card** — Weekly/bi-weekly/monthly view of saved hours
- **Calendar** — Visual month view of worked days
- **History** — Filter and export all records to CSV
- **Settings** — Hourly rate, pay period, OT thresholds (synced across app)
- **Offline** — Works without internet after first load
- **Private** — All data stored in browser localStorage, never sent anywhere
